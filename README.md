## 📘 Description

The **AI Sales Agent** is an enterprise-grade email intelligence solution designed to enhance sales productivity using **Google Gemini 3 (Vertex AI)** and **Power Automate**.

This solution automatically analyzes incoming email conversations, extracts key business context, identifies action items, and generates structured, human-readable summaries in HTML format. It enables sales teams to quickly understand ongoing discussions, prioritize follow-ups, and drive faster decision-making without manually reviewing long email threads.

Built as a lightweight, scalable API using Google Cloud Functions, the system seamlessly integrates with existing enterprise workflows, including Outlook, Power Automate, and downstream systems such as CRM platforms.

### Key Value

- Reduces time spent reading lengthy email threads  
- Highlights actionable insights and ownership clearly  
- Improves response time and sales efficiency  
- Integrates easily into enterprise automation workflows  
- Provides consistent, structured communication summaries  

This project serves as a foundational component for building a broader **AI-driven Sales Enablement Platform**.
